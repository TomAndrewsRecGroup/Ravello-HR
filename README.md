# Ravello HR

Next.js website for Ravello HR — lead generation platform with 3 conversion funnels, interactive scorecards, and sales psychology baked in throughout.

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Vercel deployment
- MDX for content

## Funnels
- **A**: Smart Hiring System (frustrated hiring leaders)
- **B**: PolicySafe™ (compliance-first SMEs)
- **C**: DealReady People™ (M&A / transformation)

## Getting Started

```bash
npm install
npm run dev
```
